## Installation <a name="installation"></a> :
* Make sure you've already installed `expo-cli` :
  ~~~ 
  npm install -g expo-cli
  ~~~
* Install package :
  ~~~
  npm install
  ~~~
* Link package :
  ~~~
  react-native-links
  ~~~
* Start project :
  ~~~
  npm start
  ~~~
